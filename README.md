# Тестовое задание Mindbox

Данное приложение является тестовым, для прохождения собеседования в Mindbox

Страница на [Vercel](https://mind-box-test-two.vercel.app)

## Структура проекта

```bash
./
├── README.md
├── package.json
├── package-lock.json
├── tsconfig.json
├── vite.config.ts
├── db.json
├── src/
│   ├── App.tsx
│   ├── App.test.tsx
│   ├── index.css
│   ├── main.tsx
│   ├── components/
│   │   └── ui/
│   ├── lib/
│   ├── shared/
│   ├── stores/
│   ├── types/
└── public/
```

## Использование

```bash
npm run start
```

## Тестирование

```bash
npm run test
```
